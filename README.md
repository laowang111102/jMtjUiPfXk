# Python学生成绩管理系统 python970

## 项目概述

本项目是基于Python语言开发的学生成绩管理系统，采用Django框架与MYSQL数据库构建，旨在为学校提供一个便捷、高效的成绩管理解决方案。

## 技术栈

- Python
- MYSQL
- Django
